# 중고 거래 게시판 MVC 패턴 사용
MVC 패턴을 사용하여 게시판을 작성하고 게시판 기본 기능인 CRUD를 동작 할 수 있도록 한다.

# 사용 언어
 - JSP
 - javascript
 - Mysql
 - HTML,CSS,Bootstrap
 
 #  특징
  - 사용자는 회원가입을 통해 회원이 될 수 있습니다.
  - 사용자는 게시판을 작성,수정,삭제 할 수 있습니다.
  - 게시판의 내용은 제목,작성자,상품 상태, 가격, 위치, 내용으로 이루어 집니다.
  - 회원은 자신의 작성글 목록을 볼 수 있습니다.
  - 사용자간 쪽지를 주고 받을 수 있습니다.
